# Caso 2 – Gestión de Prospectos en Odoo

## Pasos realizados
- Creación de un usuario interno con permisos de CRM y Contactos
- Uso del módulo CRM para la gestión de prospectos
- Registro manual de prospectos con datos reales
- Registro del RUC en el campo de notas internas
- Seguimiento del estado del lead mediante el pipeline estándar de Odoo

## Módulos utilizados
- CRM
- Contactos

## Cómo replicar la configuración
1. Crear una base de datos en Odoo (versión trial)
2. Crear un usuario con acceso a CRM
3. Ingresar al módulo CRM
4. Registrar prospectos manualmente
5. Gestionar el estado del prospecto desde el pipeline
